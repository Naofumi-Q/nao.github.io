GitHubのメモ

よく使うコマンド

        ・リモートリポジトリの内容を手元のPCにクローンする: git clone "urlをここに書く git@github.com:your_repository_name"
        　これによりリポジトリ名と同じディレクトリが作成され、このディレクトリ内ではgitコマンドが使用できる
        ・最新のリモートリポジトリの内容をローカルリポジトリに反映させる: git pull origin
        ・ローカルリポジトリの変更内容をリモートリポジトリに反映させる:
          git add newfile_name
          git commit -m "修正内容の記述"
          git push origin
